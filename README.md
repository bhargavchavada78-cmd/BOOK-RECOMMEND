📚 Book Recommendation System

A Machine Learning based Book Recommendation System developed using Python, Flask, Scikit-learn, HTML, and CSS.

The application recommends books similar to a user's favorite book by analyzing book information using TF-IDF Vectorization and Nearest Neighbors algorithms.

🚀 Features

- Search books by title
- Get similar book recommendations
- Machine Learning recommendation engine
- Fast and user-friendly interface
- Modern Black & Red UI Design
- Flask-based web application

🛠️ Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML5
- CSS3

🧠 Machine Learning Concepts

- TF-IDF Vectorization
- Content-Based Recommendation
- Nearest Neighbors Algorithm
- Text Similarity Analysis
